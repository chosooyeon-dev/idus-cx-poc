# 작품 문의·상담

## Q. 주문 전 작품 상담이 필요합니다.

아이디어스 앱/웹 내 검색 기능을 통해 구매하시려는 작품을 선택할 수 있습니다.

작품의 제작부터 배송까지 작가님이 담당하고 있어 관련된 모든 요청과 상담은 작품 상세 페이지 내 '작품문의' 버튼을 이용해 주세요.

* 작품 상세페이지에서 문의하기
  - 앱 : 오른쪽 하단 "작품 문의" 동그란 버튼
  - 웹 : 구매하기 버튼 아래의 "작품 문의" 버튼

## Q. 주문에 대한 사진과 문구는 어떻게 전달하나요?

작품의 제작부터 배송까지 작가님이 담당하고 있어 관련된 모든 요청과 상담은 작가님에게 전달이 필요합니다.

작품 상세 페이지 내 "작품 문의" 버튼 혹은 주문 내역의 '메시지' 문의 버튼을 이용해 주세요.

* 주문 내역에서 문의하기
  - 앱 : 오른쪽 하단 내정보 > 주문 배송 > 작품 선택 > 하단의 '메시지' 버튼
  - 웹 : 오른쪽 상단 내정보 > 주문 내역 > '메시지' 버튼

## Q. 주문에 대한 상담이 필요합니다.

작품의 제작부터 배송까지 작가님이 담당하고 있어 관련된 모든 요청과 상담은 작가님에게 전달이 필요합니다.

작품 상세 페이지 내 "작품 문의" 버튼 혹은 주문 내역의 '메시지' 문의 버튼을 이용해 주세요.

* 주문 내역에서 문의하기
  - 앱 : 오른쪽 하단 내정보 > 주문 배송 > 작품 선택 > 하단의 '메시지' 버튼
  - 웹 : 오른쪽 상단 내정보 > 주문 내역 > '메시지' 버튼

## Q. 작가님 연락처를 확인하고 싶어요.

작가님과는 아이디어스 메시지를 통해서만 상담이 가능합니다. 문의 및 요청 사항이 있다면 메시지를 남겨주세요. (주문 내역 > '메시지' 문의 버튼)

작가님이 작품을 제작중에 있다면 실시간 답변이 어려울 수 있습니다. 메시지를 남겼으나 하루 이상 장시간 답변이 없다면 아이디어스로 알려 주세요.

* 아이디어스 채팅상담
  - 앱 내정보 > 자주묻는질문/고객센터
  - 웹 홈페이지 하단 고객센터
* 상담 가능 시간 : 평일 오전 10시~오후 6시
