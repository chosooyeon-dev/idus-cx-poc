# 회원 정보

## Q. 비밀번호를 어디서 변경하나요?

이메일 로그인 방법으로 가입하였다면, 로그인 화면에서 재설정 가능합니다.

* 앱 : 로그인 화면의 기존 회원 로그인 > 아이디/비밀번호를 잊어버리셨나요?
* 웹 : 로그인 화면의 아이디/비밀번호 찾기

가입 시 이용하신 전화번호 또는 이메일로 비밀번호 재설정이 가능합니다.

SNS 연동 로그인 방법으로 가입한 경우, 해당하는 SNS 서비스로 접속하여 비밀번호 변경 또는 찾기를 진행해 주세요.

## Q. 아이디를 변경할 수 있나요?

이메일 로그인 방법으로 가입하였다면 아래 방법으로 변경 가능합니다.

* iOS 앱 : 내 정보 > 톱니바퀴 > 내 계정 정보 > 이메일 주소 [변경하기]
* Android 앱 : 내 정보 > 톱니바퀴 > 내 계정 정보 이메일 주소 [변경하기]
* Web : 내정보 > 회원 정보 관리 > 이메일 주소 [변경하기]

만약 변경하려는 메일 주소가 이미 가입된 계정이라면, 변경이 불가합니다. 다른 이메일을 입력해 주세요.

※ SNS 연동 방법으로 가입하였다면 이메일 주소 변경 시 더이상 해당 SNS 연동 로그인이 불가할 수 있습니다.

※ SNS 계정을 통해 회원 가입 했으나, 이메일 로그인 방법으로 변경을 희망한다면 아이디어스로 연락해 주세요.

## Q. 개명으로 회원 정보 내 이름을 변경하고 싶어요.

계정 정보 변경 페이지에서 간편하게 변경이 가능합니다.

* iOS 앱 : 내 정보 > 톱니바퀴 > 내 계정 정보 > 닉네임
* Android 앱 : 내 정보 > 톱니바퀴 > 닉네임
* Web : 내정보 > 회원 정보 관리 > 닉네임
